# Mini Project Setup

1. `npm init -y`

2. `npm install typescrpt --save-dev`

3. `npx tsc --init`

4. make `src` and `dist` directory

5. `npx tsc --watch`

6. `npm install lite-server --save-dev`

7. in `package.json` add to `scripts` `"start": "lite-server"`

8. `npm run start`

9. visit http://localhost:3000/
